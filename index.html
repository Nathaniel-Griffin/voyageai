🔐 Google Sign-In

Paste your Google OAuth 2.0 Client ID below to enable real Google Sign-In on the login page. This is optional — email login works without it.
How to get a Client ID (5 minutes):
  1. Go to Google Cloud Console → Credentials
  2. Click Create Credentials → OAuth 2.0 Client ID → Web application
  3. Under Authorized JavaScript origins, add:
    http://localhost:8080 (or whatever port you're using)
    http://127.0.0.1:8080
  4. Copy the generated Client ID (looks like 123456789-abc...apps.googleusercontent.com)
  5. Paste it below and click Save
Stored only in your browser. Clear it anytime to disable Google Sign-In.

🤖 AI Model

VoyageAI uses Mistral — a free, open-source large language model — served through the Pollinations.ai public API. No API key or signup is required. Trips are generated in real-time from your inputs.
Privacy note: Your inputs (trip type, budget, special requests) are sent to the Pollinations.ai API to generate your trip. No personal data (name, email) is transmitted. If the API is unavailable, VoyageAI falls back to a local smart-matcher that still uses your text input.
